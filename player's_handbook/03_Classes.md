# Chapter 3: Classes

In the blood-soaked ruins of Nosgoth, where the Wheel of Fate grinds souls to dust, characters rise as avatars of power, defiance, or despair. The Classes of Legacy of Kain: Bloodbound RPG embody mythic archetypes, each tied to Nosgoth’s fractured history. From the relentless fury of Blood Knights to the entropy-wielding Hylden Warlocks, every class carries a legacy that shapes their abilities and narrative destiny.

This chapter details the eight core classes. For Universal and Corrupted Perks, see **Chapter 4: Perks**.

## 3.1 Class Overview

Each class reflects a unique aspect of Nosgoth’s mythos. Players select a class at character creation, defining their core abilities, attributes, and role.

| Class | Role | Core Attributes |
| :--- | :--- | :--- |
| **Blood Knight** | Tank / Melee Bruiser | Fury, Blood |
| **Soul Reaver** | Phase Warrior / Hybrid Caster | Soul, Fury |
| **Shadowmancer** | Stealth / Controller | Shadow, Soul |
| **Sangromancer** | Blood Mage / Control | Blood, Soul |
| **Glyphwright** | Support / Utility Caster | Soul, Will |
| **Dreadblade** | Assassin / Burst Damage | Shadow, Fury |
| **Warden of Balance** | Support / Fate Caster | Will, Soul |
| **Hylden Warlock** | Dark Mage / Debuffer | Soul, Will |

## 3.2 Class Progression

*   **Starting Stats**: 5–6 skill points.
*   **Skill Point Gain**: +1 skill point per level.
*   **Attribute Growth**: Increase one Core Attribute by +1 at Levels 4, 8, 12, 16, and 20 (Max 5).
*   **Combat Bonus**: +1 die to attack rolls at Levels 5, 10, 15, and 20 (cumulative, max +4). Does not apply to skill checks.
*   **Weapon Damage Scaling**: All weapons and damaging abilities gain +1 base damage at Levels 5, 10, 15, and 20 (cumulative, max +4). A base 3-damage sword deals 7 at Level 20.
*   **Class Perks**: Gain one Class Perk every level (1–20). At **Levels 5, 10, 15, and 20**, choose between two options — each represents a different build path for that class.
*   **Universal Perks**: Gain one Universal Perk at Levels 1, 3, 5, 6, 8, 10, 11, 13, 15, 16, 18, and 20. (See Chapter 4).
*   **DV Cap**: A character's DV from attributes, armor, and perks cannot exceed 6. Temporary cover and active defense reactions can push effective DV higher.
