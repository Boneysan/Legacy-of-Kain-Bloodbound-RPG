## 📖 Quick Reference

### Reactions (1 per round)

| Reaction | Pool | When |
|---|---|---|
| **Dodge** | Shadow + Evasion | Attacked in melee or ranged |
| **Parry** | Fury + Weapon Mastery | Attacked in melee |
| **Counterspell** | Soul + Glyphcasting | Targeted by spell |
| **Opportunity Attack** | Normal attack pool | Enemy leaves your reach |
| **Spectral Shift** | Soul + Spectral Navigation | Avoid attack by phasing |

### Save Types

| Save | Attribute(s) |
|---|---|
| Blood Save | Blood |
| Will Save | Will |
| Soul Save | Soul |
| Evasion Save | Shadow + Evasion |
| Fury Save | Fury |

### Damage Types vs Armor

| Type | Armor Interaction |
|---|---|
| **Physical** | Normal armor reduction |
| **Elemental** | Armor halved (round down) |
| **Force** | Bypasses armor entirely |
| **Spectral** | Ignores physical armor |
| **Radiant** | Normal armor; ×2 vs Undead |
| **Entropic** | Normal armor; damage is unhealable |
