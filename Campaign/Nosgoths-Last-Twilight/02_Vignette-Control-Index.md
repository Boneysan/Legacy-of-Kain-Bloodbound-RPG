## Recommended Use

1. Run the vignette from the arc's VTT_Vignettes folder.
2. Open the matching arc flow matrix and apply the branch that matches the party's choice.
3. Update the arc VTT state tracker immediately.
4. Pull the listed encounter, pressure scene, or examination branch before the next session beat.
5. If you are in Arc I or Arc II and need exact downstream NPC posture, clock pressure, and encounter routing, use the detailed matrix rather than reconstructing it from the vignette and choice guides.
