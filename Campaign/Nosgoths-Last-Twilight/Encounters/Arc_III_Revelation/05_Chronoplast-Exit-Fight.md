# Chronoplast Exit Fight

- Arc: Revelation
- Recommended levels: 13-15
- Target TV: 19 Fracture package, 13 Wheel package, 23-29 reinforced
- Difficulty: Pressure
- Use when: Arc III should end on commitment, leverage, and escape pressure rather than a last simple boss duel.

## Location

- Site: Extraction route out of the Chronoplast, archive lift, or collapsing sequence bridge back toward the Material
- Terrain: Unstable bridges, collapsing futures, and one last omen-lane that points toward Convergence only if the party hold it long enough
- Pressure: Whoever controls this route controls what truth, witness, or interpretation actually leaves the machine

## Encounter Objective

- Leave the Chronoplast with the chosen leverage, witness, or version of the truth intact while the opposing bloc tries to define the arc's final meaning.

## Running Notes

- Use either the Fracture package or the Wheel package by default, not both together.
- Add the Mirror Wraith only if the exit itself is carrying the session's main combat weight.
- Let exhaustion, falling routes, and collapsing futures do as much work as the roster.
- End on a committed direction toward Arc IV, not on a fully resolved ideological victory.
