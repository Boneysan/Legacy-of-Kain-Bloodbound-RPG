# Encounter Index

Use this folder when you want a ready-to-run fight, pursuit, or pressure scene without digging back through the full campaign draft.

Each encounter file includes:

- The arc and level band it is built for
- The exact encounter location and terrain
- The monster roster with printed TVs
- The Monster Manual chapter file for each creature
- The scene objective and the main complication to emphasize

## GM Use

- Keep the current arc's session-prep file open beside the encounter sheet.
- Treat the monster source line as the authoritative stat-block location.
- If you scale an encounter up or down, preserve the location logic first and the exact TV second.
