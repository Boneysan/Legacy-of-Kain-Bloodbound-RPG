## GM Use

- Keep the current arc's session-prep file open beside the encounter sheet.
- If you want the session-by-session decision tree first, start with the relevant Encounter Guide above and then pull the specific encounter file from this index.
- If the guide mentions a variant, trigger condition, or package-selection table that the encounter file does not yet support directly, check [../03_Vignette-Encounter-Improvement-Plan.md](../03_Vignette-Encounter-Improvement-Plan.md) before improvising.
- Arc III's manipulation-first branch now has its own chase file. Use Arc_III_Revelation/06_Curator-Thess-Pursuit.md when the conflict leaves the main claim chamber and becomes a deeper-archive hunt.
- Treat the monster source line as the authoritative stat-block location.
- If you scale an encounter up or down, preserve the location logic first and the exact TV second.
- Baseline encounter TVs assume a 4-PC party near the lower or middle part of the listed level band and no more than one meaningful allied combatant acting at full strength.
- If the party is above the encounter's midpoint level or brings 2 or more strong allied combatants, treat the reinforced option or a roughly 25% TV increase as the default starting point.
- Many Arc III and Arc IV pressure scenes intentionally sit below standard TV because terrain, timing, witnesses, and ideological stakes are carrying part of the difficulty budget.
