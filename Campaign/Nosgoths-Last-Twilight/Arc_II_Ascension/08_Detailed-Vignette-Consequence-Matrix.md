## Arc II Closing Check

- If `fortress_claim` is still unstable, bring that instability north as a person, order, or debt. Do not leave it behind as backstory.
- If `sarafan_bloc: one_contact` survives Arc II, spend it once before Arc III hardens the split or erases it.
- If `hylden_engagement` is `negotiated`, the party now owns both a technical unlock and a political stain. Treat both as true in every following scene.
- Finalize Arc_II_VTT-State.md before Arc III starts. Arc II is wide enough that forgotten carry-forward details will flatten the Chronoplast handoff if they are not written down.
