Use this as the fast control sheet for Awakening. Run the vignette, update Arc_I_VTT-State.md, then pull the listed encounter or branch before the next prep block. For per-choice downstream detail, see 08_Detailed-Vignette-Consequence-Matrix.md.

## Fast Use

- If a vignette creates `public_story: panic` or `open_truth`, check the Purge-at-First-Light branch before assuming the party gets a quiet descent or exit.
- If a vignette creates `reliquary_status: examined`, make the next cult move target the recognized PC, witness, or weakest observer rather than the box itself.
- If a vignette creates `serit_trust: earned`, spend it immediately in warning beats, volunteered Accord context, or better route knowledge. Do not leave that advantage abstract.
